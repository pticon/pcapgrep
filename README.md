pcapgrep
========

grep like in multiple pcap files.

Usage
-----
```
usage: pcapgrep [options] <pattern> <pcap1 ... pcapN>
options:
        -h           : display this and exit
        -v           : display version number and exit
        -e <pattern> : use <pattern> as the pattern (can be used multiple times)
```
